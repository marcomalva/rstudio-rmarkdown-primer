# Installed Linux Packages

I keep this file updated until I get around to create a Dockerfile with my customized RStudio setup.

## Package List

The package list is save here: [dpkg-query-packagelist.txt](./dpkg-query-packagelist.txt)

That file was generated with from within the container with:

```bash
sudo dpkg-query -f '${binary:Package}\n' -W  > dpkg-query-packagelist.txt
chown rstudio:rstudio dpkg-query-packagelist.txt
```

It can be replayed with:

```bash
sudo xargs -a installedpackages apt install
```

See also:

- [How can I see DPKG and apt history](https://linuxhint.com/see-dpkg-apt-history/)

## Connect to Container

```bash
$ podman exec -it rstudio_mounted /bin/bash
```
